# Network Design and Subnetting

This section of the project focuses on designing the initial enterprise network for UnoSecurity.

Key tasks completed:

• Planned the network topology for multiple departments  
• Performed subnetting to allocate IP ranges for each network  
• Calculated network addresses, broadcast addresses, and usable hosts  
• Designed the logical network structure for future routing implementation  

Tools used:

Cisco Packet Tracer  
TCP/IP Subnetting  

Files included:

- Packet Tracer topology file
- Assignment documentation explaining subnet calculations
