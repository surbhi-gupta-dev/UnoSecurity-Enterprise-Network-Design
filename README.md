# UnoSecurity Enterprise Network Design (Cisco Packet Tracer)

This project was completed as part of my networking coursework where I designed and implemented a simulated enterprise network for a company called UnoSecurity.

The project demonstrates core networking concepts used in real IT infrastructure environments.

## Key Concepts Implemented

• Network subnetting and IP address planning  
• Static routing configuration  
• Dynamic routing using RIP and OSPF  
• VLAN segmentation and Inter-VLAN routing  
• EIGRP routing protocol  
• Access Control Lists (ACL) for traffic filtering  

## Tools Used

Cisco Packet Tracer  
TCP/IP Networking  
Routing Protocols (RIP, OSPF, EIGRP)  
VLAN and ACL Configuration

## Repository Structure

01-Network-Design-Subnetting  
02-Static-Routing-DHCP  
03-Dynamic-Routing-RIP-OSPF  
04-VLAN-EIGRP-ACL  

Each folder contains the Packet Tracer topology files, configuration files, and documentation explaining the implementation.
