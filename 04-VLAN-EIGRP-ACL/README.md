# VLAN, EIGRP and Access Control Lists

This section implements network segmentation and traffic filtering.

Key tasks completed:

• Created VLANs to segment departmental networks  
• Configured inter-VLAN routing to enable communication between VLANs  
• Implemented EIGRP dynamic routing protocol  
• Configured Access Control Lists (ACL) to restrict and control network traffic  

Security rules implemented include:

• Guest network allowed access only to web server  
• HR host allowed to access accounting server  
• DNS access restricted to specific VLANs  

Concepts demonstrated:

VLAN configuration  
Inter-VLAN routing  
EIGRP routing protocol  
ACL traffic filtering

- Packet Tracer topology
